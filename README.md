## 프로젝트 개요

- **프로젝트명**
    
    Shoppy
    
- **목적**
    
    쇼핑몰 웹사이트 클론 코딩
    
- **기능**
    
    
    - 어드민 페이지 구현
      - 어드민 페이지에서 상품 추가 가능

    - 장바구니 기능 구현
      - 디테일 페이지에서 장바구니로 저장 가능
      - 장바구니 페이지에서 개수 변경 및 삭제 가능
      
    - 리액트 hooks를 활용하여 react-query를 hooks에서 관리함
    
    - Context를 사용하여 로그인 여부 확인
